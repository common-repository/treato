A snapshot of the Treato healthcare search engine: usable insights from aggregated patient experiences written in forums and blog posts.

== Description ==

Embed Treato healthcare search engine into your WordPress sidebar.

= About Treato =

[Treato](https://treato.com) travels around the web to collect and analyze patient experiences written in forums and blog posts. Our patented semantic technology aggregates the posts to generate useful insights about users' medications, diseases and symptoms - all from other patients.

With over 2 billion collected posts, our users receive insight about side effects, drug comparisons, which drugs people feel work, and much more.

https://www.youtube.com/watch?v=8rEXhLXfOWM

= Treato WordPress Plugin =

With this plugin you can add a sidebar widget which allows you and your users to search drugs see what patients are saying about different drugs.

= More Information =

Visit [Treato.com](https://treato.com) to learn more about what this plugin can provide you. You can also see it live on our [blog](http://blog.treato.com/). For any questions or more information, please [contact us](http://corp.treato.com/contact).

== Installation ==

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
2. Search for "Treato".
3. Install the plugin by pressing the "Install" button.
4. Activate the plugin by pressing the "Activate" button.
5. Navigate to "Appearance" -> "Widgets"
6. Drop the treato widget into your sidebar.

= Minimum Requirements =
* WordPress version 3.5 or greater.
* PHP version 5.2.4 or greater.
* MySQL version 5.0 or greater.

= Recommended Requirements =
* The latest WordPress version.
* PHP version 7.0 or greater.
* MySQL version 5.6 or greater.

== Frequently Asked Questions ==

= What are the widget proportions on the sidebar? =

Treato sidebar widget is 300x350 pixels, which fits in most sidebars. We are working on fitting it into sidebars narrower than 300 pixels.
